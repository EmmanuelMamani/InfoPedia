¿Que es HTML?

HTML significa Lenguaje de marcas de hipertexto, sirve para el desarrollo de paginas web,no es un lenguaje de programacion, no es necesario tener un editor de texto para realizar archivos HTML sin embargo es recomendable contar con alguno ya que fcilita la manera de trabajo en esta pagina se unilizara Sublime Text como editor de texto,no es dificil aprender solo se requiere un poco de practica y voluntad.

Etiquetas fundamentales

Todo documento html costa de tres etiquetas indispensables sin las cuales no se podria hacer una pagina de manera adecuada estas etiquetas son:

  • < html >
  • < head >
  • < body >

Html es la etiqueta base donde se contendra todas las demas etiquetas

Head esta etiqueta contiene informacion que el cliente no ve pero es importante para el navegador

Body en esta etiqueta se colocara todo el contenido que queramos que tenga nuestra pagina

Etiquetas para hacer texto

Existen varias etiquetas para hacer texto depende a las necesidades que tengamos,entre ellas tenemos:

  • < b >
  • < strong >
  • < i >
  • < blockquote >
  • < cite >

< b > pone en negrita todo el contenido que hay dentro de la etiqueta

< strong > al igual que la letra anterior pone el texto en negrita pero ademas el contenido dentro de la etiqueta sera usado para que el navegador lo tome en cuenta para sugerir la pagina

< blockquote > nos sirve para hacer frases o citar textos

< cite > se utiliza para escribir el autor de una cita o de algun articulo

Como insertar una imagen y un video

Al momento de querer insertar imagenes o videos debes tener mucho cuidado con los derechos de autor ya que no se quiere tener problemas legales por lo tanto es recomendable hacer tus propios archivos

< img > en esta etiqueta se añade la ubicacion de la imagen puede ser una ubicacion local como una ubicacion de internet , tambien esta etiqueta cuenta con distintos atributos como se muestra en la imagen

< video > se debe añadir la ubicacion del video como en la etiqueta de img

Como añadir un enlace

Para añadir un enlace utilizamos la etiqueta < a > pero esta etiqueta tiene parametros que deben ser explicamos para ello adjuntamos una imagen

Como hacer una lista

Las listas son recursos necesaios en las paginas nos sirven para hacer menus, indices, etc. Existen dos tipos de listas:

  • < ol > lista ordenada
  • < ul > lista no ordenada

Estas etiquetas nos sirven para crear listas pero requerimos de otra etiqueta para añadir elementos a las listas esa etiqueta es < li >

Como hacer tablas

las tablas tienen como proposito la creacion de columnas de texto que cada uno lo puede usar a su conveniencia, nos puede servir para hacer cuadros comparativos, distribuciones, etc. Para hacer una lista primero debemos crear su estructura para ello usaremos la etiqueta < table > dentro de ella podremos añadir filas con la etiqueta < tr > y dentro de esa etiqueta podemos añadir columnas con la etiqueta < td > esto puede ser un poco xonfuso por lo tanto se dejara una imajen que demuestre su funcionamiento y otra de como se ve implementado en una pagina

Estructura de la pagina

La estructura de pagina permite una mayor comprension acerca de la pagina para el navegador y para aquella persona que tenga que editar la pagina posteriormente de ser necesario, esta estructura se enfoc en la etiqueta < body > y se da de la siguiente manera:

< header > esta etiqueta nos servira de contenedor para el encabezado de la pagina contendra el titulo,logo o imagen relacionada(opcional),un menu o indice del contenido

< h1 > aqui va el titulo de la pagina

< div > aqui va la imagen o logo ,se lo coloca en esta etiqueta solamente por estetica

< nav > aqui va el menu que lo podemos colocar como una lista para una mejor apreciacion, se lo coloca en la etiqueta nav para que el buscador lo tome encuenta como palabras claves o importantes

< selection class="wrappper" > esta es la etiqueta contenedora de las secciones principal y la seccion secundaria de nuestra pagina

< selection class="main" > en esta etiqueta van los articulos o articulos importantes de la pagina para hacer un articulo se debe utilizar la etiqueta < article > est sera contenedor de subtitilos (< h2 >) y de parrafos (< p>)

< aside > aqui van los articulos o contenido secundario, tambien puede ir publicidad

< footer > este es el pie de pagina por lo general aqui va enlaces de contacto como ser numero telefonico o perfiles de redes sociales y tambien podria ir derechos de autor u otra informacion legal

Metadatos

los metadatos es informacion importante para el navegador ayuda a que encuentre la pagina o que sugiera nuestra pagina dependiendo a nuestro contenido se dara a conocer dos metadatos los cuales son:

< meta name="description" content="......" > esto nos permite añadir una pequeña descripcion de lo que trata nuestra pagina se debe colocar la descripcion en la parte punteada

< meta name="keywords" content="......" > esto nos permite añadir palabras clave para que nos sugiera el buscador se deben agregar las palabras en la parte punteada y separarlas por ","
